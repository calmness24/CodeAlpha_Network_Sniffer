# CodeAlpha Network Sniffer

## DDescription
This project is a basic network sniffer built using Python and Scapy. It captures live network packets and displays packet information such as source IP, destination IP, protocol type, and ports. 

## Features
- Captures network packets
- Displays source and destination IP addresses
- Detects TCP and UDP protocols
- Displays source and destination ports

## Technologies Used
- Python
- Scapy

## Run the Project

Install Scapy:

pip install scapy

Run: 

sudo python3 sniffer.py
